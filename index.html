    • Rb-ftp

    FTP client and server implementations in realbasic

    Project maintained by charonn0 Hosted on GitHub Pages — Theme by mattgraham

    RB-FTP aims to provide easy-to-use FTP client and FTP server socket classes for REALstudio applications.

    How This Project Is Organized

    This project implements three main classes:

    FTPSocket

    This class provides the SuperClass for the other FTP classes. It manages both the control connection and the data connection for both client and server implementations.

    FTPClientSocket

    Inherits FTPSocket, provides a FTP client. This class provides a number of methods named after the FTP verb they perform. For example, to upload a file the STOR method is called.

    FTPServerSocket

    Inherits FTPSocket, provides a FTP server. This class is incomplete, I would say it's about 50% of the way to working.

    Project Status

    At the moment, the FTPClientSocket class works and can upload and download files, change directory, etc. The only big thing left on the client socket is parsing LIST responses, which would probably go somewhere in the FTPSocket.DataAvailableHandler method which handles the DataAvailable event of the data connection.

    Work on the FTPServerSocket class is still underway.
